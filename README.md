# 🌸 PetalTalk (가칭)

“감정을 꽃으로 표현하는 서비스”

꽃말을 단순히 보여주는 것을 넘어,  
사용자가 자신의 감정을 꽃과 메시지로 자연스럽게 전달할 수 있도록 돕는 것을 목표로 합니다.

---

## ✨ 주요 기능

### 1. 꽃 검색 및 꽃말 조회

사용자가 꽃 이름을 검색하면 해당 꽃의 의미(꽃말)를 확인할 수 있습니다.

- 꽃 이름 기반 검색
- 꽃말 및 간단한 설명 제공
- 직관적인 UI로 빠른 정보 탐색

---

### 2. 감정 기반 꽃 추천

사용자가 자신의 감정이나 상황을 입력하면, 그에 어울리는 꽃을 추천합니다.

- 예: 고백, 위로, 축하, 짝사랑 등
- 감정에 맞는 꽃 3~5개 추천
- 각 꽃의 의미와 추천 이유 제공

---

### 3. 꽃말 기반 메시지 생성

선택한 꽃을 기반으로 감성적인 메시지를 자동 생성합니다.

- 짧은 문구 / 긴 문장 선택 가능
- 다양한 톤 (감성, 담백, 캐주얼 등) 지원
- 카드나 편지에 바로 활용 가능

---

### 4. 공유용 카드 생성

꽃과 메시지를 조합해 공유 가능한 이미지를 생성합니다.

- SNS 공유용 카드 UI
- 꽃 + 메시지 + 배경 디자인 자동 구성
- 간편한 다운로드 및 공유 기능

---

## 🎯 서비스 방향

PetalTalk은 단순한 “꽃말 사전”이 아니라,  
사용자의 감정을 꽃을 통해 표현하고 전달할 수 있도록 돕는  
감정 기반 인터랙션 서비스입니다.
